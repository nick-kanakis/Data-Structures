# Theory of Arrays

## Time Complexity
| Access | Search | Insert (middle/start) | Delete (middle/start) | Insert (end) | Delete (end) |
|:-------:|:-------:|:-------:|:-------:|:-------:|:-------:|
| O(1) | O(n) | O(n) | O(n) | O(1) | O(1) |

## Space Complexity
`O(n)`

## Operations
- `void add(Item)`
- `void insert(Index, Item)`
- `void remove(Index)`
- `Item find(Index)`
- `Boolean empty()`
