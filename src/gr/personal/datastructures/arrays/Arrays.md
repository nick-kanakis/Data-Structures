# Theory of Arrays

## Time Complexity
| Access | Search | Insert (middle/start) | Delete (middle/start) | Insert (end) | Delete (end) |
|:-------:|:-------:|:-------:|:-------:|:-------:|:-------:|
| O(1) | O(n) | O(n) | O(n) | O(1) | O(1) |

## Space Complexity
`O(n)`

## Operations
- void add(item)
- void insert(index, itme)
- void remove(item)
- void delete(index)
- item find(index)
- Boolean empty()
